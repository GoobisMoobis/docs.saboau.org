# What is **MiraOverloaded**?
**MiraOverloaded** is a Town of Us: Mira extension mod, created because the Among Us modscape has a problem; People are always craving more content than is available in base Mira.

When MiraOverloaded[^1] first started development, there were only a few widely-used extension mods, namely [TouMiraRolesExtensionMod](https://github.com/rewalo/TownOfUsMiraRolesExtension)[^2], and [Chaos Tokens](https://github.com/xChipseq/ChaosTokens). REM is an extremely stable mod, having survived multiple Mira updates, and Chaos Tokens has updated to support the latest versions of Mira and fix bugs. Both are still actively maintained, however specifically with REM, it isn't updated often, because it doesn't need to be.

REM is the grandparent of all role extension mods. Without it as a pioneer, more recent extension mods would have never come about. However, the lack of updates with REM also means that it doesn't get new features often. It's a meaningful addition to a regular Mira game, and REM is 100% the most widely-used extension. But some people, including Ninja and Goober[^3], wanted to do three things.
1. Create an even larger extension mod than REM with more features/updates
2. Make it the largest extension mod ever made while maintaining code readability and minimizing bugs
3. Give it enough roles to fill the craving people have for extra content in Mira

This is part of the reason that MiraOverloaded directly integrates the Chaos Tokens mod into itself by default[^4].
